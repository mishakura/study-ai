# Study AI

Personal repository for studying Artificial Intelligence and Machine Learning.

## Structure

| Folder | Description |
|--------|-------------|
| `fundamentals/` | Math, statistics, linear algebra, probability |
| `ml-basics/` | Classic ML — scikit-learn, pandas, regression, classification, clustering |
| `deep-learning/` | Neural networks, CNNs, RNNs, PyTorch/TensorFlow |
| `nlp/` | Transformers, LLMs, embeddings, text processing |
| `projects/` | Hands-on mini projects |
| `papers/` | Paper summaries and implementations |
| `courses/` | Notes organized by course |
